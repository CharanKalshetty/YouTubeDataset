This is a simple program for practicing data analysis and data analytics on YouTube data.

The main data is stored in 'YouTube Dataset.csv'. 'youtubedata.txt' has the same data as of 'YouTube Dataset.csv', this is
the file which is impported as a DataFrame in python.

The program is stored in 'Program.py' and also 'code.text'. Each block of code is explained by comments.

The other csv files are some of the output generated by 'Program.py' python file.
